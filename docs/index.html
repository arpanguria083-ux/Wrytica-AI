FAQ

Common questions

Is Wrytica AI really free?

Yes — 100% free. The app is open source. You only need a free Google Gemini API key (or use Ollama/LM Studio locally for completely free offline use). There are no usage limits imposed by Wrytica.

Do I need to create an account?

No. Wrytica AI has zero accounts, zero login, zero tracking. Download, install, add your API key in Settings, and start. That's it.

How does the privacy work exactly?

When you use Gemini, your text goes directly from your machine to Google's API — Wrytica never sees it. When you use Ollama or LM Studio, nothing leaves your device at all. Your API key is stored in your browser's localStorage, not on any server.

What AI models does it support?

Currently supports Google Gemini (free tier available at Google AI Studio), Ollama (run any open-source model fully offline), and LM Studio (local model runner with a GUI). More providers are on the roadmap.

What's the difference between Free and Pro?

Free includes all core writing tools — paraphrase, grammar, summarize, citations, OCR, and AI chat. Pro (coming soon) will add unlimited knowledge base storage, batch processing, DOCX/PDF export, team guardrails, and priority support. Join the waitlist for 50% off.

Is it available on Mac or Linux?

Currently Windows only in v1.0. Mac and Linux builds are on the roadmap. Join the waitlist and let us know your platform — it directly influences what we build next.
