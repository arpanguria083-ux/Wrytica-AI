# 🔑 How to Get Your Gemini API Key for Wrytica AI

## Quick Setup (2 minutes)

### 1. **Get Your Free API Key**
1. Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the generated API key

### 2. **Add to Wrytica AI**
1. Open Wrytica AI in your browser
2. Go to **Settings** (gear icon in sidebar)
3. Select **"Online (Gemini)"** provider
4. Paste your API key in the **"Gemini API Key"** field
5. Click **"Test Connection"** to verify it works

### 3. **Start Using AI Features**
- ✅ Paraphrasing with tone analysis
- ✅ Grammar checking with forecasting
- ✅ Text summarization
- ✅ Citation generation
- ✅ AI chat assistant

## 🔒 Privacy & Security

- **Your API key stays local** - stored only in your browser
- **Never shared** - we don't send your key to our servers
- **You control usage** - your key, your Google AI Studio quota
- **Free tier available** - Google provides generous free usage

## 💡 Tips

- **Free Tier**: Google AI Studio provides substantial free usage
- **Usage Tracking**: Monitor your usage in Google AI Studio
- **Multiple Keys**: You can create multiple keys for different projects
- **Revoke Anytime**: Delete keys in Google AI Studio if needed

Don't want to use an API key? Switch to local LLMs:
